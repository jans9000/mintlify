- title: Konten erstellen

Neue Benutzer zu Ihrer Organisation hinzufügen

Das erste Benutzerkonto, das eine Organisation von Sensolus erhält, ist immer ein Konto mit Organisationsadministrator-Berechtigungen. Dieser Benutzer verfügt über alle erforderlichen Berechtigungen, um alle Einstellungen zu verwalten und zusätzliche Benutzer einzuladen.

Um neue Benutzer einzuladen, gehen Sie zu

<Frame caption="">
  <img src="https://cloud.sensolus.com/documentation/images/icons/icon-admin-section.png" />
</Frame>

Admin → Zugriffsverwaltung → Benutzer und beginnen Sie, neue Benutzer zu erstellen oder sie per E-Mail einzuladen.

<Frame caption="">
  <img src="https://cloud.sensolus.com/documentation/images/userinterface/users_organisations/users_createnewuser.png" />
</Frame>

Als Administrator können Sie so viele Benutzer hinzufügen, wie Ihr Produktplan erlaubt. Sie können verschiedene Arten von Benutzern erstellen, indem Sie die Rolle für den neuen Benutzer auswählen, und jedes Profil hat einen anderen Satz von Benutzerberechtigungen. Je nach dem Team, das die Sensolus-Plattform nutzt, können Sie neue Benutzerkonten erstellen und festlegen, welche Berechtigungen sie erhalten.

Standardmäßig werden den Organisationen folgende Rollen zur Verfügung gestellt:

1. **Organisationsadministrator**: Diese Rolle hat die meisten Berechtigungen. Benutzer mit dieser Rolle haben die Berechtigung, alle Elemente in ihrem Konto zu erstellen, zu ändern und zu löschen. Sie können neue Benutzer erstellen, neue Tracker hinzufügen, bestehende Benutzer löschen, Warnungen und Geozonen erstellen usw.
2. **Organisationseditor**: Diese Rolle hat weniger Berechtigungen als der Administrator. Benutzer mit dieser Rolle können ihre eigene Rolle verwalten, die Mitglieder und Tracker der Organisation, zu der sie gehören, sehen und Geozonen und Warnungen erstellen. Ein Benutzer mit Editor-Rechten kann keine neuen Benutzer erstellen oder die Einstellungen bestehender Benutzer ändern oder diese löschen.
3. **Organisationsbetrachter**: Diese Rolle hat weniger Rechte als der Administrator und der Editor. Benutzer mit Betrachter-Rechten können nur überwachen, was im Konto passiert, sie können keine neuen Elemente erstellen oder bestehende Elemente ändern. Benutzer mit dieser Rolle können ihre eigene Rolle verwalten, Benutzerinformationen der Mitglieder ihrer Organisation einsehen, die Tracker der Organisation, zu der sie gehören, sehen, aber sie können keine Geozonen und Warnungen erstellen.

<Note>
  Note: It is also possible to create custom user roles. This is a more advanced feature if you need specific user groups to have a specific set of permissions on the platform. It comes with the Enterprise Security add-on.
</Note>

Wenn Sie Änderungen an Ihrem eigenen Benutzerkonto vornehmen möchten, können Sie Ihre persönlichen Einstellungen bearbeiten. Dort können Sie ein neues Bild hinzufügen, Ihr Passwort ändern usw.

Um auf Ihre persönlichen Einstellungen zuzugreifen, klicken Sie auf den Benutzernamen oder das Benutzerprofilbild in der Symbolleiste in der oberen rechten Ecke des Bildschirms und wählen Sie

<Frame caption="">
  <img src="https://cloud.sensolus.com/documentation/images/icons/icon-user.png" />
</Frame>

Persönliche Einstellungen.
