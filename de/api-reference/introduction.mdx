## Willkommen

Es gibt zwei Möglichkeiten, API-Dokumentation zu erstellen: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) und [MDX-Komponenten](https://mintlify.com/docs/api-playground/mdx/configuration). Für das Starter-Kit verwenden wir die folgende OpenAPI-Spezifikation.

## Authentifizierung

Alle API-Endpunkte werden mit Bearer-Tokens authentifiziert und aus der Spezifikationsdatei entnommen.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
