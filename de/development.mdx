<Info>
  **Voraussetzung**: Bitte installieren Sie Node.js (Version 19 oder höher), bevor Sie fortfahren.<br />
  Bitte aktualisieren Sie auf `docs.json`, bevor Sie fortfahren, und löschen Sie die veraltete `mint.json` Datei.
</Info>

Befolgen Sie diese Schritte, um Mintlify auf Ihrem Betriebssystem zu installieren und auszuführen:

**Schritt 1**: Installieren Sie Mintlify:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

**Schritt 2**: Navigieren Sie zum Docs-Verzeichnis (wo sich die `docs.json` Datei befindet) und führen Sie den folgenden Befehl aus:

```bash
mintlify dev
```

Eine lokale Vorschau Ihrer Dokumentation wird verfügbar sein unter `http://localhost:3000`.

### Benutzerdefinierte Ports

Standardmäßig verwendet Mintlify Port 3000. Sie können den Port, auf dem Mintlify läuft, mit dem `--port` Flag anpassen. Um Mintlify beispielsweise auf Port 3333 auszuführen, verwenden Sie diesen Befehl:

```bash
mintlify dev --port 3333
```

Wenn Sie versuchen, Mintlify auf einem bereits belegten Port auszuführen, wird der nächste verfügbare Port verwendet:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify-Versionen

Bitte beachten Sie, dass jede CLI-Version mit einer bestimmten Version von Mintlify verknüpft ist. Wenn Ihre lokale Website nicht mit der Produktionsversion übereinstimmt, aktualisieren Sie bitte die CLI:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## Validierung von Links

Die CLI kann bei der Validierung von Referenzlinks in Ihrer Dokumentation helfen. Um fehlerhafte Links zu identifizieren, verwenden Sie den folgenden Befehl:

```bash
mintlify broken-links
```

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Error: Could not load the &#x22;sharp&#x22; module using the darwin-arm64 runtime">
    Dies kann auf eine veraltete Version von Node zurückzuführen sein. Versuchen Sie Folgendes:

    1. Entfernen Sie die aktuell installierte Version von Mintlify: `npm remove -g mintlify`
    2. Aktualisieren Sie auf Node v19 oder höher.
    3. Installieren Sie Mintlify neu: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
    Solution: Go to the root of your device and delete the \~/.mintlify folder. Afterwards, run `mintlify dev` erneut.
  </Accordion>
</AccordionGroup>

Neugierig, was sich in der CLI-Version geändert hat? [Schauen Sie sich das CLI-Änderungsprotokoll an.](https://www.npmjs.com/package/mintlify?activeTab=versions)
